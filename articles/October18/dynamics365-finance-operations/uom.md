#  <a name="unit-of-measure-conversion-for-variant"></a>バリアントの測定単位換算

[!include[banner](../../includes/banner.md)]

この機能は、倉庫管理システム全体を通じて、独自の現物分析コードと測定単位 (UOM) 換算を製品バリアント レベルで構成できるようにするものです。 現時点では、少数のシナリオしかサポートされていません。
