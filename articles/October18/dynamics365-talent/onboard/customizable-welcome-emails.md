#  <a name="customizable-welcome-emails"></a>カスタマイズ可能なようこその電子メール





管理者は、オンボード時に新しい採用者に送信されるようこそ電子メールをパーソナライズできます。 これにより、個別に対応されている感じが強くなり、電子メール応答率が向上します。
