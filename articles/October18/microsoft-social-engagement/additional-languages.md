#  <a name="additional-languages"></a>追加言語

[!include[microsoft-social-engagement banner](../includes/microsoft-social-engagement.md)]

- バハサ語
- チェコ語
- ヒンディー語
- ハンガリー語 
- 韓国語

- ヒンディー語

- セルビア語
