# <a name="improved-app-discovery-for-consumers"></a>コンシューマー向けのアプリの探索の向上




アプリ利用者は、PowerApps の統一ランディング ページを通じて自分および自分の組織のモデル駆動型アプリとキャンバス アプリを検索、プレビュー、実行して、素早く生産性を高めることができます。
