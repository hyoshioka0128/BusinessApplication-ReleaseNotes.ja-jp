# <a name="dynamics-365-mobile-support-email-login"></a>Dynamics 365 Mobile が電子メール ログオンをサポート


[!include[banner](../../includes/banner.md)]

Dynamics 365 Online または PowerApps のモデル駆動型アプリのユーザーは、サーバー URL ではなく、自分の電子メール アドレスを使用して、Android および iOS の Dynamics 365 アプリにサインインできます。 サインインした後、ユーザーは利用可能なすべてのオンライン組織と環境のアプリを表示してアクセスできます。
