# <a name="sharing-enhancements-for-canvas-apps"></a>キャンバス アプリの共有の拡張




キャンバス アプリの共有が簡単になります。アプリによって使用されるデータ ソースをユーザーが表示できるため、そのデータにアクセスするためのユーザー アクセス許可を個別に設定しやすくなります。
