# <a name="powerapps-supports-intune"></a>PowerApps が Intune をサポート




Android と iOS 上の PowerApps アプリが Intune と統合され、モバイル アプリケーション管理に対して Intune ポリシーをサポートするようになったので、お客様のデータに対する制御とセキュリティが向上します。
