# <a name="formula-formatting-easier-to-read"></a>読みやすくなった式の書式




式バー エディターの機能強化により、アプリ開発者は、式のフォーマットを行い、それを読み取って理解することができます。 アプリ開発者がエンドユーザー向けにより複雑な基幹業務用アプリケーションを作成するなかで、このフォーマット機能により、複数の開発者はアプリで共同作業を行えるようになります。
