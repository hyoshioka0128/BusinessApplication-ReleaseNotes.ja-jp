# <a name="create-entities-and-apps-in-the-same-tool"></a>同じツールでのエンティティとアプリの作成


[!include[banner](../../includes/banner.md)]

アプリを作成または変更するとき、別のサイトを開くことなく PowerApps Studio でエンティティおよびオプション セットを作成および変更できます。
