# <a name="localization-support-for-hololens-2"></a>HoloLens 2 でのローカライズのサポート

Dynamics 365 Layout は、HoloLens 2 の次のロケールで使用できるようになりました。 

- 米国
- カナダ (英語およびフランス語)
- 英国
- オーストラリア
- ドイツ
- フランス
- 中国
- 日本
- 台湾 (音声サポートなし)
- 香港特別行政区 (音声サポートなし)
