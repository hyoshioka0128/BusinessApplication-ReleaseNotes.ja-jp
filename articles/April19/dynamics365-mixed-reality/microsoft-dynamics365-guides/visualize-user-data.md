# <a name="visualize-process-efficiency-with-dashboards"></a>ダッシュボードでのプロセス効率の視覚化

Power BI ダッシュボードで提供される組み込みテレメトリを使用して、各ユーザーがタスクまたはステップを完了するのに必要な時間など、貴重なインサイトを得ることができます。 データを集約してプロセスのパフォーマンスに焦点を合わせることができます。
