# <a name="fbx-and-gltf-formats-automatically-transcoded"></a>FBX および GLTF 形式は自動的にトランスコードされる

FBX ファイルと GLTF ファイルは、ライブラリーへのアップロード時に、Microsoft HoloLens (GLB) 用の正しい形式に自動的にトランスコードされます。
